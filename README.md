# Программа сортировки слиянием нескольких файлов

## Описание:
Программа принимает на вход более одного текстового файла с данными.
Данные записаны в столбик (каждая строка файла – новый элемент). Строки могут содержать любые не пробельные
символы, строки с пробелами считаются ошибочными. Также считается, что файлы предварительно
отсортированы.

Результатом работы программы является новый файл с объединенным содержимым
входных файлов, отсортированным по возрастанию или убыванию путем сортировки слиянием.

Файл будет располагаться:

* При запуске через IDE: `.../TestTaskCFT/workDirectory/`
* При запуске через командную строку: `.../TestTaskCFT/src/workDirectory/`

Параметры задаются при запуске через командную строку или в разделе Run->Edit Configuration запуске через IDE.
## Описание структуры проекта
`src.exeptions.FileContainsUnsortedIntArray.java` - исключение, выбрасывается при отсутствии сортировки в исходном файле, приложение продолжает работу

`src.exeptions.SpaceContainsException.java` - исключение, выбрасывается при наличии пробельных символов в исходных данных файлов, приложение подолжает работу с корректными данными

`src.fileManager.fileManager.java` - класс, отвечающий за чтение файлов и запись данных в выходной файл.

`src.merge.MergeSort.java` - класс, содержит статические методы сортировки слиянием массивов

`src.merge.SubSortedArray.java` - класс, содержит статические методы по проверки сортированности массивов, методы возвращают наибольший сортированный массив


`src.workDirectory` - директория для работы с файлами через командную строку, в неё помещаются исходные данные и там же появляется выходной файл

`src.Main` - отправная точка программы, работает с аргументами командной строки

`workDirectory` - директория для работы с файлами IDE, в неё помещаются исходные данные и там же появляется выходной файл

### При написании использовалось:
* SDK: 19 java version "19.0.1"
* Language level: SDK default

# Иструкция по запуску через командную строку:
1. Установить Java Development Kit на свой компьютер с официального сайта: https://www.oracle.com/java/technologies/downloads/#jdk19-windows
2. Разархивировать zip файл на свой компьютер
3. Поместить файлы с входными данными в папку: `.../TestTaskCFT/src/workDirectory/`
4. Открыть терминал/командную строку для папки `...TestTaskCFT/src/`
5. Ввести команду `javac Main.java`

6. Для запуска программы потребуются аргументы:
Запуск производится командой `java Main`
Далее указываются параметры по порядку:
    * режим сортировки (-a или -d), необязательный, по умолчанию сортируем по возрастанию;
    * тип данных (-s или -i), обязательный;
    * имя выходного файла, обязательное;
    * остальные параметры – имена входных файлов, не менее одного. 

Примеры запуска программы:

```java Main -a -i out.txt file1.txt file2.txt file3.txt``` - производтся сортировка файлов с числовыми данными по возрастания из 3х файлов

```java Main -i out.txt file1.txt file2.txt file3.txt``` - эквивалентно предыдущему,

```java Main -d -s out.txt file1.txt file2.txt file3.txt``` - производтся сортировка файлов со строковыми данными по убыванию из 3х файлов
